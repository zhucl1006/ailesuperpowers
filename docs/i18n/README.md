# i18n（繁體中文化）說明

本目錄定義本倉庫的繁體中文化規範、術語與驗收方式。

## 目標

- 讓使用者可見內容以繁體中文為主。
- 保護程式執行結構（frontmatter、指令、路徑、程式碼片段）。
- 降低翻譯造成的格式與行為回歸。

## 文件

- `docs/i18n/terminology-zh-Hant.md`：術語表
- `docs/i18n/translation-rules.md`：翻譯規則

## 驗收清單

- [ ] `name:`frontmatter 欄位值同步翻譯
- [ ] 命令列與程式碼區塊未被破壞
- [ ] Markdown 鏈接語法完整
- [ ] Hook/Plugin 輸出結構不變
- [ ] 關鍵測試可執行並通過
