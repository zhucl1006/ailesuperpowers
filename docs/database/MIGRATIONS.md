## 3. 移民政策

當儲存/佈局行為改變時：

1. 將遷移理由記錄在`RELEASE-NOTES.md`
2. 更新安裝和平臺文檔
3. 提供明確的舊路徑→新路徑指導
4. 包括 Windows 的後備/故障排除說明（如果適用）
